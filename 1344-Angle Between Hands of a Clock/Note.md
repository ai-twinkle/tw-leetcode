# 1344. Angle Between Hands of a Clock

Given two numbers, `hour` and `minutes`, return the smaller angle (in degrees) formed between the `hour` and the `minute` hand.

Answers within `10^-5` of the actual value will be accepted as correct.

**Constraints:**

- `1 <= hour <= 12`
- `0 <= minutes <= 59`

## 基礎思路

本題要求計算時鐘上「時針」與「分針」之間所形成的較小夾角（以度數表示），並允許 `10^-5` 以內的誤差。解題關鍵在於將兩根指針各自的角度位置轉換成可直接相減的數值，再從圓周的兩段弧中取出較小的一段。

在思考解法時，可掌握以下核心觀察：

- **每根指針的角速度為固定常數**：
  分針每分鐘移動 6 度；時針每小時移動 30 度，並隨分鐘累積額外的移動量，因此時針位置同時受到小時與分鐘影響。

- **兩指針位置可合併為單一線性式**：
  將時針角度與分針角度相減後，整體會化簡為一個僅與小時、分鐘相關的式子，無須分別計算兩個中間角度再相減。

- **時鐘為環狀，存在兩段互補弧**：
  任意角度差所對應的兩段弧加總恆為 360 度，題目要的是其中較小的一段。

- **符號不影響夾角大小**：
  指針差值的正負僅代表方向，與夾角大小無關，因此可直接取絕對值。

依據以上特性，可以採用以下策略：

- **以化簡後的公式直接求出兩指針的帶符號差值**，避免額外的中間運算。
- **取絕對值以消除方向影響，得到原始夾角**。
- **若原始夾角超過 180 度，則以 360 度減去之**，即可得到較小的那一段弧。

此策略全程僅以常數次運算完成，簡潔且高效。

## 解題步驟

### Step 1：將兩指針位置合併為單一帶符號差值

利用化簡後的公式，把時針與分針的位置摺合為一個帶符號的差值；其中以 `hour % 12` 處理 12 點等同 0 點的情況，省去分別計算兩指針角度的中間值。

```typescript
// 將兩指針的位置摺合為單一帶符號差值，省去中間變數
const signedDifference = (hour % 12) * 30 - minutes * 5.5;
```

### Step 2：取絕對值以得到原始夾角

差值的正負僅代表方向，與夾角大小無關，因此直接取絕對值即可得到原始夾角；以 `Math.abs` 一步處理正負號，較分支判斷更為省成本。

```typescript
// Math.abs 比分支判斷更省成本，可一步處理正負號
const rawAngle = Math.abs(signedDifference);
```

### Step 3：取兩段弧中較小者作為答案

由於時鐘以 360 度為一圈，原始夾角與其互補弧加總恆為 360 度；若原始夾角大於 180 度，代表它是較大的那一段，需以 360 度減去之以取得較小弧，否則直接回傳。

```typescript
// 時鐘以 360 度為一圈，因此兩段弧中較小者即為答案
return rawAngle > 180 ? 360 - rawAngle : rawAngle;
```
